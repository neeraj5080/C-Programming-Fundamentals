# C Programming Fundamentals

This repository contains fundamental C programs that I practiced as part of my
B.Tech Computer Science Engineering coursework and self-learning.

The goal of this repository is to build a strong foundation in C programming
concepts that are essential for understanding Data Structures and advanced
programming.

## Programs Included

- Prime Number Check  
- Prime Numbers in a Given Range  
- Leap Year Check using Ternary Operator  
- Greatest of Three Numbers using Ternary Operator  
- Reverse of a Number and Palindrome Check  
- Rightmost Digit of the Integral Part of a Floating Point Number  

## Concepts Covered

- Conditional statements (if-else, ternary operator)  
- Loops  
- Functions  
- Number logic  
- Type casting  
- Basic problem-solving in C  

## Language Used
- C
